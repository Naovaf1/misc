# วิธีทำให้ไฟล์นี้เปิดผ่านลิงก์เว็บได้ถาวร

โปรเจกต์นี้เป็น static website ดังนั้นสามารถโฮสต์ฟรีได้ง่ายมาก

## วิธีที่แนะนำ: GitHub Pages

ข้อดี
- ฟรี
- ลิงก์ค่อนข้างถาวร
- แชร์ให้คนอื่นเปิดได้ทันที
- เหมาะกับไฟล์ HTML + รูปภาพแบบโปรเจกต์นี้

ลิงก์ที่ได้จะมีหน้าตาประมาณนี้

`https://USERNAME.github.io/REPOSITORY_NAME/`

โดยลิงก์หลักจะเปิด `PZEM_Technical_Summary.html` ให้อัตโนมัติผ่าน `index.html`

## ขั้นตอน

1. สร้าง repository ใหม่บน GitHub
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo
3. ไปที่ `Settings > Pages`
4. ที่ `Build and deployment`
5. เลือก `Deploy from a branch`
6. เลือก branch `main` และโฟลเดอร์ `/ (root)`
7. กด Save
8. รอ GitHub สร้างลิงก์เว็บ

## ไฟล์สำคัญ

- `index.html` ใช้เป็นหน้าเริ่มต้นของเว็บ
- `PZEM_Technical_Summary.html` คือหน้าหลักที่ต้องการให้เปิด
- ไฟล์ `.png` ทั้งหมดต้องอัปโหลดไปด้วย เพราะหน้าเว็บอ้างอิงรูปแบบ relative path

## ทางเลือกอื่น

### Netlify
- ง่ายมาก
- ลากทั้งโฟลเดอร์ไป deploy ได้
- ถ้าสมัครบัญชีและผูก site ไว้ ลิงก์จะคงอยู่

### Cloudflare Pages
- ฟรี
- เร็ว
- เหมาะถ้าต้องการผูก custom domain ในอนาคต

## ข้อควรรู้

- ลิงก์จะ "ถาวร" ตราบเท่าที่บัญชี/รีโพ/ไซต์ยังไม่ถูกลบ
- ถ้าต้องการโดเมนสวย ๆ เช่น `pzem.yourname.com` ต้องมี custom domain เพิ่ม
- ตอนนี้โปรเจกต์พร้อมสำหรับ static hosting แล้ว
